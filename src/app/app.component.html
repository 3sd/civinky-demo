Civinky Pug and Inky for CiviMail

Converts a Pug template, writen with Inky tags to inline html ready for CiviMail. Include your own CSS, JSON, and CiviMail tokens.

Try it out below and install the CiviCRM extension.

The big idea

Email is hard to get right, especially when all you have is a wysiwyg editor.

Thankfully there are better tools are out there. So lets glue them together and make them available for CiviCRM users.

In this case, the glue is a simple webservice (Civinky) that creates email friendly HTML and a CiviCRM extension to make that service available in CiviCRM.

Get started

You're welcome to use this site to create your email friendly HTML. If you get bored of copying and pasting, try installing the Civinky extension and writing Inky based emails in CiviCRM. To get started:

  1. Install the extension
  2. Set up the webservice
  3. Send some email!
